## 📌 Sobre o projeto
Uma plataforma de marketplace para compra e venda de produtos vintage, para colecionadores e apreciadores de itens clássicos, raros e retrô.

## Funcionalidades 
* Cadastro e Login
* Carrinho de produtos
* Barra de pesquisa
* Categorias de produtos 

## Tecnologias 
* Front-end: HTML, CSS, Javascript
* Back-end: Javascript
* Versionamento: Git e Github
